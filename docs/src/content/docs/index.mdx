## Features

<CardGrid stagger>
  <Card title="Moderation" icon="warning">
    <Badge text="Popular" variant="tip" /> Ban, mute, kick, and warn users. Set up anti-flood protection and word blacklists.
  </Card>
  <Card title="Automation" icon="rocket">
    <Badge text="Smart" variant="note" /> Welcome messages, CAPTCHA verification, auto-approve join requests, and keyword filters.
  </Card>
  <Card title="Content" icon="document">
    <Badge text="Core" variant="note" /> Save notes, set rules, create filters, and manage pinned messages with ease.
  </Card>
  <Card title="Multi-Language" icon="translate">
    <Badge text="4 Languages" variant="default" /> Built-in support for English, Spanish, French, and Hindi — with easy extensibility for more.
  </Card>
  <Card title="High Performance" icon="seti:go">
    <Badge text="Go-Powered" variant="success" /> Built in Go with concurrent processing, Redis caching, and PostgreSQL storage.
  </Card>
  <Card title="Self-Hostable" icon="setting">
    <Badge text="Docker Ready" variant="caution" /> Deploy your own instance with Docker, binaries, or build from source.
  </Card>
</CardGrid>
