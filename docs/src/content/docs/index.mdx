## Features

<CardGrid stagger>
  <Card title="Moderation" icon="warning">
    Ban, mute, kick, and warn users. Set up anti-flood protection and word blacklists.
  </Card>
  <Card title="Automation" icon="rocket">
    Welcome messages, CAPTCHA verification, auto-approve join requests, and keyword filters.
  </Card>
  <Card title="Content" icon="document">
    Save notes, set rules, create filters, and manage pinned messages with ease.
  </Card>
  <Card title="Multi-Language" icon="translate">
    Built-in support for English and Spanish, with easy extensibility for more languages.
  </Card>
  <Card title="High Performance" icon="seti:go">
    Built in Go with concurrent processing, Redis caching, and PostgreSQL storage.
  </Card>
  <Card title="Self-Hostable" icon="setting">
    Deploy your own instance with Docker, binaries, or build from source.
  </Card>
</CardGrid>
